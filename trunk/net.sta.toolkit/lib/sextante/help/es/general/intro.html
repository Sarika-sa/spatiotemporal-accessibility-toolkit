Introducción

Bienvenido a este tutorial de SEXTANTE. SEXTANTE es un conjunto de algoritmos de análisis geográfico de código libre desarrollado para la Junta de Extremadura para satisfacer sus propias necesidades de análisis de la información espacial. Este proyecto sigue la línea de trabajo de la Junta de Extremadura en el desarrollo de programas libres, que comenzó con el exitoso proyecto LinEx.

La librería de SEXTANTE esta integrada en algunos de los más populares SIG Java de escritorio. Si tú tienes una aplicación SIG y quieres incorporar SEXTANTE como plataforma de análisis espacial este tutorial es para tí.

La información con la descripción de los algoritmos de SEXTANTE no se encuentran en este libro. Para encontrar esta información se debe descargar la ayuda contextual.

SEXTANTE es software libre y se distribuye bajo licencia GPL. Si encuentras algo en él y en este manual que creas mejorable, no dudes en hacerlo saber. Vuestra ayuda es un elemento fundamental para conseguir un manual cada vez más útil y preciso.

Elementos básicos de SEXTANTE

Existen cuatro elementos básicos en la interfaz gráfica de SEXTANTE, los cuales permiten el acceso a todas las herramientas de formas diversas, en función de las necesidades de cada usuario y cada proyecto de trabajo.

Dependiendo de la aplicación SIG que se esté usando, estos elementos pueden estar disponibles a través del menú (generalmente debajo del menú llamado SEXTANTE) o la barra de herramientas como se muestra a continuación.

  • Gestor de extensiones de SEXTANTE. Es el elemento principal de la interfaz gráfica de la librería. Permite ejecutar una extensión o correr una proceso por lotes basado en un algoritmo. Esta será la forma mas habitual en que utilices SEXTANTE.

  • Modelizador gráfico de SEXTANTE. El modelizador te permitirá combinar varios algoritmos gráficamente, creando un proceso mayor que engloba varios subprocesos (algoritmos).

  • Línea de comandos de SEXTANTE. La consola de SEXTANTE permitirá a los usuarios más avanzados hacer un uso mas ágil del programa y automatizar tareas mediante la creación de sencillos scripts.

  • Historial de SEXTANTE. Todas las acciones llevadas a cabo con SEXTANTE se registran en un historial. Éste puede posteriormente consultarse, así como ejecutar desde él sistemáticamente dichas acciones, facilitando la repetición de procesos.
