This zip file contains the SEXTANTE library for geospatial analysis. 
This distribution is targeted at developers wanting to incorporate 
geospatial analysis capabilities to their software, using SEXTANTE.

It contains the following folders.

|-core: core SEXTANTE files, including geoalgorithms, base classes and graphical components
|-bindings: bindings for several applications and libraries, structured in subfolders. 
|	    A readme file for each of them is included
|-help:help files for context help


To start working with SEXTANTE, check out the SEXTANTE programming guide from or website.

For more information: http://www.sextantegis.com
